# Bufdir.no Dokumentasjon

Dette er hovedsiden for dokumentasjon av Bufdir.no-løsningen. Her finner du detaljert informasjon om teknisk arkitektur og strategier for monitorering.

## Innhold

### [Løsningsarkitektur](./Architecture/architecture.md)
Dette dokumentet beskriver den overordnede arkitekturen for Bufdir.no. Det gir en oversikt over systemets moduler (både hovedmodul og støttemoduler), teknologistakk, dataflyt, integrasjoner og fellesmønstre for infrastruktur og sikkerhet.

### [Monitorering og overvåking](./Monitorering/monitoring.md)
Dette dokumentet beskriver den helhetlige strategien for monitorering av Bufdir.no-økosystemet. Det dekker bruk av Azure Monitor og OpenTelemetry, oppsett av distribuert sporing (distributed tracing), varslingskonfigurasjon, kostnadsestimater og spesifikke monitoreringsplaner for de ulike modulene.
